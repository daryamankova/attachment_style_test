# Карта ваших отношений

Одностраничный интерактивный тест Дарьи Маньковой про стиль привязанности, близость, слияние и автономию в отношениях.

## Файлы

- `index.html` — загрузчик сайта.
- `chunks/` — части основного HTML с тестом, вопросами, расчетом и расшифровками.

Сайт рассчитан на запуск через веб-сервер или GitHub Pages.
